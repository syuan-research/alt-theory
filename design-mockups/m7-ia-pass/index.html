Variant B

What is the same in both

  • Pane logic: left = navigation, center = conversation, right = conversation-derived detail (collapsed by default), settings = first-level gear.
  • Settings: icon tab list, model config at the top, participant panel with label + sharing default.
  • Researcher mode = user mode + workbench left pane, review as a full page, advanced tabs on the right rail.
  • Mode indicator + one-line posture copy in the header.
  • Approval card with real scope disclosure, shown inline in the sample conversation.
