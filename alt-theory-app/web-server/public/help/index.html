Alt Theory Trial Help

Use this page if you are testing Alt Theory during the IAPS presentation.

Quick Start

  1. Open test.alttheorylab.com.
  2. Sign in with the Account ID and Login Code on your trial card.
  3. Type a question in the input box at the bottom of the screen.
  4. Press Enter or tap the send button.
  5. After testing, open the feedback survey from your trial card and enter the same Account ID.

Knowledge Base Setting

The current switch is labeled Use EP knowledge base. It is above the message input area.

  • Keep it on for environmental psychology theory, concepts, and literature-related questions.
  • Turn it off for general conversation, writing help, or questions unrelated to environmental psychology.

Private Session Setting

This page is for prototype trial access only. Using this tool during the conference does not mean formal research participation or informed consent.

The default trial mode may create a research-readable test session. If you do not want your test session to be read, turn on Private session before testing.

  • Default: research-readable test session.
  • Private session: not read by anyone and deleted after 7 inactive days.

If Something Gets Stuck

  • If sign-in fails, check that the Account ID and Login Code match your card exactly.
  • If the input box is disabled, wait a moment and reload the page.
