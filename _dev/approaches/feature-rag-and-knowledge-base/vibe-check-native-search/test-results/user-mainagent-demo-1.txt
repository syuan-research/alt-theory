  连接分析：Perception → Place

  Heft 用 20 年完成了从个体知觉到集体地方的跨越，核心桥梁：

  1997（感知端）: Gibson 的 affordance 是 个体-环境 的关系性概念。"surface affords sitting-on"
  取决于身高相对于台面。导航不是认知地图，而是对 vista 过渡的时间性感知。

  2018（地方端）: 整合 Barker 的 behavior settings — 地方不是物理围合，而是通过相互依赖的联合行动 +
  环境支持涌现的动态系统（如课堂、周二晚上的扑克局）。

  四个连接机制:

  1. 个体→集体: 1997 的 affordance 是个体感知；2018 的 behavior settings 扩展到集体-环境系统
  2. 功能→规范: 1997 的 "expedient action"（效率驱动）→ 2018 的 "proper
  action"（文化规范驱动），地方因此有了道德维度
  3. 时间性扩展: 1997 导航研究证明知觉跨越时间；2018 证明参与也跨越时间（日常惯例创造归属感）
  4. 结构感知: 2014 point-light display 实验证明人能仅从动作模式识别 behavior settings — affordance
  感知从物理层面延伸到社会层面

  有意思的观察: 这个文件有 core+details 合并的格式问题（line 52 开始重复了 frontmatter），但反而提供了 Midwest
  实验和 point-light 实验的实证数据，对深度分析有利。子代理也发现了这一点。

  回答用户

  现在我能回答那个研究生了。关键洞察是：不是没有理论能解释，而是主流理论关注的是个体生理/认知恢复，而你的体验核心是
  关系性和社会性的。

  能解释你的体验的理论：

  1. Relational Restoration Theory (Hartig 2021) — 最
  fit。你和朋友晚上吃饭不是在"恢复注意力"，而是在恢复关系资源（trust, shared history, mutual understanding）。RRT
  说：环境通过提供 "distance from ordinary demands" + "affordances for mutually appreciated activities"
  来促进关系恢复。小吃街虽然破旧，但它提供了这种 distance 和 affordance — 脱离了工作环境，可以自在地聊天吃饭。
  2. Place Attachment — PPP Framework (Scannell & Gifford 2010) —
  你对"家门口的破旧小吃街"的感情不是因为它"好看"，而是因为它承载了个人经历（Individual Person）+ 社会关系（Social
  Place Attachment — 在这里和朋友吃饭）+ 连续性（Continuity — 过去到现在的时间线）。PPP 框架明确说：attachment
  不需要物理环境"好"，甚至 Physical Place Attachment 可以是"place
  dependence"——这个地方提供你需要的资源（吃饭+社交），不需要它漂亮。
  3. Behavior Settings (Wicker 1987) — 小吃街本身就是一个 behavior
  setting：在特定时空边界内，人们和物（摊位、桌子）互动，执行"夜宵社交"这个功能。Wicker 强调 settings
  是动态的社会建构 — 它的价值不在于物理美观，而在于"standing arrangements"和"negotiated
  order"。摊主和常客之间的默契就是 negotiation。

  不能很好解释你的体验的理论（你直觉觉得不对的原因）：

  4. Supportive Design (Ulrich 1991) — 这就是你说的"理论说要好看要有植物"。Ulrich 的框架基于 evolutionary
  predispositions（进化准备性），认为人类对自然环境有生物学层面的偏好。但这个理论是针对医疗环境（patients,
  hospitals）的，它的核心是个体生理应激恢复。你的小吃街体验不是"降低生理应激"，而是"增强社会连接" —
  这是完全不同的恢复路径。
  5. ART / SRT — 同理。它们解释的是个体从"定向注意力疲劳"或"生理应激"中恢复。你没觉得注意力疲劳需要恢复，你觉得的是
  "下班很晚之后和朋友在一起很开心" — 这是 relational/collective 层面的。

  核心差距：主流恢复理论是 individual-psychophysiological 的，你的体验是 relational-social 的。

  Hartig 2021 的 RRT 和 CRT 正好填补了这个 gap — 它们把恢复从个体扩展到了关系和集体层面。但它们还很新（2021），而且
  目前主要研究自然环境，还没太多关注"城市破旧小吃街"这种场景。这可能是你"找不到理论 fit"的原因 —
  理论存在，但还没被应用到你的场景类型上。
