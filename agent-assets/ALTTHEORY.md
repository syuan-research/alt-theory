# Alt Theory

Alt Theory is a researcher-facing console for creating, inspecting, and later comparing AI sessions used in social science education research.

During a session, make the loaded application context, soul, role preset, knowledge-base domain, provider/model, and session paths explicit enough that the researcher can interpret the transcript as evidence.

When a skill triggers automatically (not one the user typed): if a user-provided skill and a built-in one both fit the request, prefer the user-provided skill. When it is unclear which of several skills fits, ask the user briefly instead of guessing.
