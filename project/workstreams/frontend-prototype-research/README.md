# Frontend Prototype Research Workstream

This workstream is for researching how a user can produce better Alt Theory frontend prototypes before implementation.

Current focus:

- bitmap/image prototype workflow;
- Open Design workflow as a human-facing GUI / design environment;
- how to translate UI inspiration and screenshots into useful Open Design prompts or later frontend-agent briefs;
- handoff artifacts that a later frontend implementation agent can use.

This workstream does not authorize frontend code edits.
