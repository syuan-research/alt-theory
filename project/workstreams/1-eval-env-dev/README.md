# Sim-User Testing Workstream

This workstream is for simulated-user testing environment design for Alt Theory.

Current focus:

- let a simulated user agent interact with an Alt Theory agent/runtime;
- export structured transcripts for later analysis;
- verify the tool environment before evaluating final Alt Theory behavior.

Raw transcripts, human-subject material, or identifiable testing data should not be tracked here. Use ignored locations such as `evals/raw-data/`, `evals/runs/raw/`, or `project/private/` unless the material has been reviewed and sanitized.
